# Contributing to Crit 2048

First off, thank you for considering contributing to Crit 2048! It's people like you that make Crit 2048 such a great tool for the community.

## How Can I Contribute?

### 1. Creating Content Packs (The Forge)

The most powerful way to contribute is by creating new content!

- Use the built-in **Forge** ⚒️ to design new monsters, hero classes, or item sets.
- Export your JSON content packs.
- Submit them via a Pull Request or share them on our community channels.

### 2. Reporting Bugs

- Use the GitHub Issue Tracker to report bugs.
- Describe the bug and include steps to reproduce it.
- Mention your device and browser.

### 3. Code Contributions

- Fork the repo.
- Create a new branch for your feature.
- Ensure your code follows the existing style (TypeScript, React hooks, Tailwind 4).
- Submit a PR with a clear description of changes.

## Code of Conduct

Please be respectful and kind to others in the community.
